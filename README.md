🛍️ متجر Balance 3D

Balance 3D هو متجر إلكتروني متكامل متخصص في بيع منتجات الطباعة ثلاثية الأبعاد، مع دعم كامل لتجربة تسوق حديثة تشمل المستخدمين والإدارة.

⸻

🚀 مميزات المشروع

👤 للمستخدمين:

* تصفح المنتجات المطبوعة ثلاثية الأبعاد
* نظام سلة مشتريات (Cart)
* إنشاء حساب وتسجيل دخول
* تتبع الطلبات
* تقييم المنتجات وكتابة مراجعات
* طلب طباعة تصميم خاص حسب رغبة العميل
* دعم كامل للغتين (العربية / الإنجليزية)

⸻

🧑‍💼 لوحة الإدارة (Admin Panel):

* إضافة وتعديل وحذف المنتجات
* إدارة الطلبات ومتابعتها
* إدارة المستخدمين
* عرض تقارير بسيطة عن المتجر
* التحكم في التقييمات (ردود / إدارة / حذف)
* إمكانية التوسع لإضافة:
    * تحديد أسعار الشحن
    * إدارة العروض والخصومات
    * إدارة المحتوى والردود

⸻

💳 الدفع الإلكتروني (مستقبلي)

المشروع مصمم ليكون جاهز لربط بوابات الدفع التالية:

* Visa
* Mastercard
* Apple Pay
* STC Pay
* مدى (Mada)
* بوابة الدفع: Moyasar

⸻

🌐 اللغة

    * العربية 🇸🇦

🇬🇧 English

🛍️ Balance 3D Store

Balance 3D is a full-featured e-commerce platform specialized in 3D printed products, offering a modern shopping experience for both customers and administrators.

⸻

🚀 Features

👤 Customer Features:

* Browse 3D printed products
* Shopping cart system
* User registration & login
* Order tracking
* Product reviews & ratings
* Custom 3D print request service
* Full bilingual support (Arabic / English)

⸻

🧑‍💼 Admin Dashboard:

* Add / edit / delete products
* Manage orders
* Manage users
* Basic analytics & reports
* Moderate reviews (reply/delete)
* Extensible features:
    * Shipping price control
    * Discounts & promotions
    * Content & feedback management

⸻

💳 Payment Integration (Future Ready)

The system is designed to support:

* Visa
* Mastercard
* Apple Pay
* STC Pay
* Mada
* Payment gateway: Moyasar

⸻

🌐 Language Support

* Full language switch support:
    * Arabic 🇸🇦
    * English 🇬🇧

⸻

⚙️ How to Run

Currently the project is a static frontend (HTML/CSS/JS)

Steps:

1. Download the project
2. Open:
index.html
3. Run directly in the browser
No server or installation required
⸻

📌 Notes

* Built using AI assistance (Claude)
* No backend integrated yet
* Payment systems are not active yet (future integration ready)
* Fully scalable into a complete e-commerce system

⸻

🛣️ Roadmap

* Add backend (database + API)
* Integrate Moyasar payment gateway
* Improve admin dashboard
* Add shipping system
* Add notifications (Email / SMS / WhatsApp)
* Performance & UI improvements

⸻

💡 Project Idea

A modern marketplace for:

* Ready-made 3D printed products
* Custom 3D printing orders based on user designs
